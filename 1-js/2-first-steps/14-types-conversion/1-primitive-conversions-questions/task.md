```js
"" + 1 + 0

true + false
6 / "3"
"2" * "3"
4 + 5 + "px"
"$" + 4 + 5 
"4" - 2 
"4px" - 2 
7 / 0 
"  -9\n" + 5
"  -9\n" - 5
5 && 2 
2 && 5 
5 || 0 
0 || 5
null + 1
undefined + 1
```
